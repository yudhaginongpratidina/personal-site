# GIT KONFIGURATION

Installation

```bash
sudo apt install git
sudo apt-get install git
yum install git
```

Check Git Version

```bash
git --version
```

Check Git Configuration

```bash
git config --list
```

Config Username

```bash
git config --global user.name "example 1"
```

Config Email

```bash
git config --global user.email example@gmail.com
```
