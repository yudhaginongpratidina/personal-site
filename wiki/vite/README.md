# VITE GUIDE

```bash
npm create vite@latest .
```

```bash
npm create vite@latest app_name -- --template name_of_template
```

See create-vite for more details on each supported template: 

- `vanilla` 
- `vanilla-ts` 
- `vue` 
- `vue-ts` 
- `react` 
- `react-ts` 
- `react-swc` 
- `react-swc-ts` 
- `preact`
- `preact-ts` 
- `lit` 
- `lit-ts`
- `svelte`
- `svelte-ts`
- `solid`
- `solid-ts`
- `qwik`
- `qwik-ts`
